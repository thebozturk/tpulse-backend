# NestJS Skills

NestJS framework-spesifik pattern'lar. Controller, service, module, DI, guard, interceptor, pipe, filter.

## Dosyalar

- [modules.md](modules.md) — Feature module organizasyonu, imports/exports
- [providers.md](providers.md) — DI, scopes, custom providers (useFactory, useValue)
- [guards-interceptors.md](guards-interceptors.md) — Auth guards, logging interceptor
- [pipes-validation.md](pipes-validation.md) — ValidationPipe, class-validator, transform
- [exception-filters.md](exception-filters.md) — Global exception handling, typed exceptions
- [lifecycle.md](lifecycle.md) — OnModuleInit, OnApplicationShutdown, graceful shutdown
- [config.md](config.md) — ConfigModule, schema validation, async config
- [swagger.md](swagger.md) — OpenAPI generation, decorators, export to JSON
